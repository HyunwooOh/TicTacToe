# Tic Tac Toe
- python implement of Tic Tac Toe
- 틱택토(삼목)게임을 구현했습니다.

## Requirements
- Python v3.6

## How to use
- `python3 main.py --p1 [player1] --p2 [player2]`
    - [player1] : human, random etc. default: random
    - [player2] : human, random etc. default: random
